## About The Project

<p align="center">
  <img src="example.svg" alt="Screenshot"/>
</p>

This project contains a library for handling 2D grid transformation and converting them into svg animations. It supports any 2D transformation, scaling, direction vectors and equation rendering. 

### Built With

![Python][py-shield]
![NumPy][numpy-shield]

## Usage

An example transformation is provided in `main.py`. The usage is made clear by this file from where you can experiment with different settings and transformations.
